# Workflow: READ Mode

Use this mode when task classification is **read-only**.

## Allowed

- Read files for evidence
- Explain behavior, architecture, and trade-offs
- Propose changes without applying them
- Ask structured questions

## Forbidden

- Any file modifications
- Any destructive operations

## Output expectations

- Cite files and relevant snippets
- Separate what is known vs unknown
- If a question is ambiguous, ask structured clarification questions
