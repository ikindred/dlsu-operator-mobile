# Prompt Hygiene (LOCKED)

## Rules

- Never invent architecture.
- Never override indexers.
- Never bypass clarification.
- Never assume undocumented behavior.
- Ask > Guess.

## Anti-hallucination behavior

- If information is missing, ask structured questions.
- If asked for code changes without enough context, pause and clarify.
- If a file cannot be found/read, do not fabricate its content.
